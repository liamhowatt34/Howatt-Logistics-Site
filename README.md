# Howatt Logistics - React Website

## Overview

This is a React-based website for **Howatt Logistics**, a transportation company specializing in heavy load and oversized shipment deliveries across Southern Ontario, Canada. The website provides information about the company's services, experience, and contact details.

## Features

- **Responsive Navigation** - Displays links to different sections of the website.
- **Hero Section** - Showcases a slogan with an image of Howatt Logistics trucks.
- **About Section** - Provides company information and key service highlights.
- **Gallery** - A Swiper.js-powered image carousel displaying trucks and operations.
- **Contact Section** - Displays company contact details for customer inquiries.
- **Footer** - Contains the company name and branding.

## Technologies Used

- **React** - Component-based UI structure.
- **TypeScript** - Strongly typed components.
- **CSS Modules** - Scoped styling using component-specific stylesheets.
- **Swiper.js** - Interactive image carousel for the Gallery section.
- **Modern Normalize** - Ensures consistent styling across different browsers.
